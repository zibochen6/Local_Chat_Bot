## 📋 NFC卡片写入步骤

### 方法1：使用手机NFC写入工具

1. **下载NFC写入应用**：
   - Android: "NFC Tools"、"NFC TagWriter"
   - iOS: "NFC Tools"（需要iOS 13+）

2. **写入步骤**：
   ```
   打开应用 → 选择"写入" → 选择"URL" → 输入产品URL → 触碰NFC卡片
   ```

### 方法2：使用电脑NFC读写器

1. **硬件要求**：USB NFC读写器
2. **软件**：ACR122U NFC读写器软件
3. **写入步骤**：
   ```
   连接读写器 → 打开软件 → 选择"写入URL" → 输入产品URL → 放置NFC卡片
   ```
